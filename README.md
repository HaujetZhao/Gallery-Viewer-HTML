# 📸 Local Photo Viewer (本地图片查看器)

一个现代、轻量且**完全本地化**的 Web 图片查看器。无需上传，保护隐私，直接在浏览器中流畅管理和查看你的本地图片库。

## ✨ 核心亮点

*   **🛡️ 隐私优先**：基于 File System Access API 构建，所有操作均在本地完成，图片数据绝不上传云端。
*   **🚀 极致便携**：单 HTML 文件入口，**支持 `file://` 协议直接打开**，无需本地服务器，随开随用。
*   **⚡ 高性能**：
    *   **虚拟滚动 & 懒加载**：轻松处理数千张图片的文件夹。
    *   **IndexedDB 缓存**：自动缓存缩略图，二次打开秒级加载。
    *   **多线程处理**：利用 Web Workers (如有) 或异步队列生成缩略图，不阻塞主界面。
*   **🎨 现代 UI/UX**：
    *   **瀑布流布局**：自适应 Masonry 网格，完美展示不同比例图片。
    *   **沉浸式查看**：支持平滑缩放、拖拽平移、键盘快捷键导航。
    *   **多主题 & 响应式**：精心设计的界面，适配各种屏幕尺寸。

## 🛠️ 功能特性

*   **📁 文件夹树状导航**：侧边栏实时同步文件系统结构，支持展开/收起。
*   **🔍 强大的文件管理**：
    *   支持文件重命名、删除（直接操作本地文件）。
    *   拖拽移动/整理文件。
    *   智能排序（按名称、大小、日期、随机）和实时搜索过滤。
*   **ℹ️ 详细的 EXIF 信息**：
    *   查看相机参数（光圈、快门、ISO 等）。
    *   **智能地图定位**：自动识别 GPS 信息，提供 **Google、高德、百度** 三种地图跳转，支持 WGS84 坐标自动纠偏。
*   **🖱️ 高效交互**：
    *   **滚动感应区**：鼠标悬停在屏幕边缘自动滚动，解放双手。
    *   **快捷键支持**：`Ctrl+C` 复制图片，方向键切换，ESC 关闭。

## 💻 技术栈

*   **Core**:原生 JavaScript (ES6+), HTML5, CSS3
*   **Storage**: IndexedDB (LocalForage 风格封装)
*   **API**: File System Access API
*   **Libraries**:
    *   `exif-js`: 元数据读取
    *   `SparkMD5`: 文件哈希计算
    *   FontAwesome: 图标库

## 🚀 快速开始

1.  下载本项目代码。
2.  直接双击 `index.html` 打开。
3.  点击界面通常的 "**打开文件夹**" 按钮，授权访问你的图片目录。
4.  Enjoy!

## ⚠️ 兼容性说明

由于使用了 **File System Access API**，本项目目前最佳支持以下浏览器（桌面端）：
*   Chrome / Edge (86+)
*   Opera

*注意：Firefox for Desktop 目前暂未原生支持此 API 的完整功能。*
